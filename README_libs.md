# Bibleotheken erstellen
Um Bibliotheken zu erstellen, erstellen Sie eine Python-Datei im `Z_libs`-Ordner. Der Name der Datei ist der Name der Bibliothek. In dieser Datei erstellen Sie eine Klasse, die genau den gleichen Namen wie die Datei hat. Jede einzelne Funktion in der Klasse wird dann als Funktion genutzt.
